# EV Car Charging Reservation System

A Java desktop application for EV owners to find, view availability, and reserve electric vehicle charging stations.

## Technologies Used
- Java
- File I/O for data persistence

## Features
- User login and registration (Customer/Admin)
- View available charging slots
- Make and cancel reservations
- Admin management of slots
- Wallet system for payments
- View booking history
- Generate report for admin
